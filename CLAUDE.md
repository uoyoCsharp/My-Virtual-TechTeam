## AI Agent Framework

This project uses a multi-agent AI development framework for structured software development.

**Framework Entry**: `.ai-agents/FRAMEWORK.md`

### Quick Commands

| Command | Agent | Purpose |
|---------|-------|---------|
| `#init` | Conductor | Initialize project |
| `#analyze` | Analyst | Analyze requirements |
| `#design` | Architect | Create architecture |
| `#implement` | Developer | Implement code |
| `#fix` | Developer | Fix bugs |
| `#review` | Reviewer | Code review |
| `#test` | Tester | Generate tests |
| `#status` | Conductor | Show progress |

### Standard Workflow

```
#analyze → #design → #implement → #review → #test
```
